# 📚 توثيق تطبيق **ادرس** - المنصة التعليمية اليمنية

---

## 🗂️ نظرة عامة

**ادرس** هو تطبيق تعليمي مخصص لطلاب المرحلة الإعدادية في اليمن، يتيح:
- قراءة الكتب المدرسية والملخصات
- تتبع التقدم الدراسي (محلياً وعبر الإنترنت)
- مساعدة ذكية بالذكاء الاصطناعي (Gemini)
- لوحة متابعة لأولياء الأمور
- لوحة تحكم للمعلمين

## 💾 قاعدة البيانات المحلية (Offline)

يستخدم التطبيق **Hive** لتخزين البيانات محلياً:

| البيانات المحفوظة | الوصف |
|------------------|-------|
| بيانات الطالب | الاسم، الرقم الأكاديمي، الصف |
| التقدم الدراسي | الصفحة الحالية، الوقت المقضي، نسبة الإنجاز |
| حالة القراءة | آخر صفحة تمت قراءتها لكل كتاب |
| بيانات ولي الأمر | الاسم، قائمة الأبناء |
| حالة المزامنة | هل البيانات تمت مزامنتها مع السيرفر |

### آلية المزامنة التلقائية:
1. يحفظ التطبيق التقدم محلياً باستمرار (حتى بدون إنترنت)
2. عند فتح التطبيق وتوفر الإنترنت، تُرسَل البيانات تلقائياً للسيرفر
3. يمكن المزامنة يدوياً بالضغط على زر 🔄 في الصفحة الرئيسية

---

## 🤖 ميزة الذكاء الاصطناعي (AI)

### طريقتا الاستخدام:

#### 1. AI للصفحة الكاملة:
- اضغط على زر **AI** الأرجواني في شريط التنقل
- سيقوم بشرح محتوى الصفحة الكاملة

#### 2. AI للنص المحدد:
- حدد أي نص في الكتاب (اضغط مطولاً)
- سيظهر زر **🤖 بحث AI** فوق النص المحدد
- اضغط عليه لشرح النص المحدد فقط

### الـ Prompt المستخدم:
```
أنت معلم متخصص في شرح المناهج الدراسية لطلاب اليمن في المرحلة الإعدادية.
1. اشرح هذا النص بطريقة مبسطة وواضحة
2. قدم أمثلة عملية إن أمكن
3. اذكر النقاط المهمة التي يجب على الطالب تذكرها
4. استخدم اللغة العربية الفصحى البسيطة
```

## 👨‍🎓 دليل الطالب

1. **اختر "طالب"** من شاشة الحسابات
2. **أدخل رقمك الأكاديمي** (مثال: 78246)
3. **الصفحة الرئيسية** تعرض:
   - تقدمك الإجمالي
   - زر "أكمل من حيث توقفت" (يفتح آخر صفحة قرأتها)
   - المواد الدراسية مع شريط تقدم لكل مادة
4. **داخل الكتاب:**
   - التنقل بين الصفحات باستخدام الأسهم
   - عرض الملخصات (ملخص 1، 2، 3)
   - استخدام AI لشرح أي محتوى

---

## 👨‍👩‍👦 دليل ولي الأمر

1. **اختر "ولي الأمر"** من شاشة الحسابات
2. **أدخل رقمك الخاص** (مثال: 78246)
3. **إضافة الأبناء:**
   - اضغط "إضافة ابن" وأدخل رقمه الأكاديمي
   - يُضاف الابن مرة واحدة فقط ويُحفظ تلقائياً
4. **عرض التقدم:**
   - اضغط على اسم الابن لعرض تفاصيل تقدمه
   - اضغط 🔄 لتحديث الدرجات من السيرفر

---

## 👨‍🏫 دليل المعلم

1. **اختر "أستاذ"** من شاشة الحسابات
2. **أدخل الرقم الوظيفي وكلمة المرور** (78246 / 123)
3. **لوحة المعلم تعرض:**
   - بيانات المعلم والمادة
   - الصفوف الدراسية مع نسبة النشاط
4. **داخل الصف:**
   - قائمة الطلاب مع حالة كل طالب (نشط / متوسط / يحتاج متابعة)
   - الضغط على طالب لعرض تقريره التفصيلي
   - تصفية الطلاب حسب الحالة

## 🔧 الـ Backend (Django)

### بنية قاعدة البيانات:
| الجدول | الوصف |
|--------|-------|
| Student | بيانات الطلاب |
| Teacher | بيانات المعلمين |
| Parent | بيانات أولياء الأمور |
| Book | الكتب المدرسية |
| BookPage | صفحات الكتب |
| PageSummary | نوع الملخص |
| PageSummaryPage | صفحات الملخص |
| Progress | تقدم الطالب |

### APIs المتاحة:
| الـ Endpoint | الطريقة | الوصف |
|-------------|---------|-------|
| `/api/subjects/` | GET | قائمة المواد |
| `/api/books/` | GET | قائمة الكتب |
| `/book-pages/{page_number}/` | GET | صفحة الكتاب (HTML) |
| `/api/summary-html/` | GET | صفحة الملخص |
| `/api/progress/` | POST | حفظ/تحديث تقدم الطالب |
| `/api/students/` | GET | البحث عن طالب بالرقم الأكاديمي |

## 🔄 تغيير السيرفر

لتغيير رابط السيرفر (IP أو دومين):

1. افتح الملف: `lib/core/config/app_config.dart`
2. غيّر قيمة `backendBaseUrl`:
```dart
// للشبكة المحلية:
static const String backendBaseUrl = 'http://192.168.1.100:8000';

// للإنترنت:
static const String backendBaseUrl = 'https://your-domain.com';
```
3. أعد بناء التطبيق: `flutter build apk --release`
