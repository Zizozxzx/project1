الحل
(أ) س / ص = { ١٤ ، ١٢ ، ١٠ ، ٨ } / { ٨ ، ٦ ، ٤ ، ٢ }
شكل (١-٥)
{ م : م ∈ س ، م ∉ ص } =
{ ٢ ، ٤ ، ٦ }
وتمثلها المنطقة المظللة في الشكل (١-٥).
(ب) ص / س = { ٨ ، ٦ ، ٤ ، ٢ } / { ١٤ ، ١٢ ، ١٠ ، ٨ }
شكل (١-٦)
قارن بين س / ص ، ص / س ماذا تلاحظ؟
ثانيًا : المجموعة المتممة
غالبًا نرمز للمجموعة الشاملة بالرمز ( ش )، فإذا كانت ش هي مجموعة طلبة فصلك، وكانت س هي مجموعة طلبة فصلك المشتركين في الإذاعة المدرسية، فإن مجموعة طلبة فصلك غير المشتركين في الإذاعة المدرسية تسمى متممة المجموعة س.
المجموعة المتممة للمجموعة س هي مجموعة العناصر التي تنتمي إلى ش ولا تنتمي إلى س، ويرمز لها بالرمز س̄، أي أن:

{ م : م ∈ ش ، م ∉ س } = س̄
