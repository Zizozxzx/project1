الوحدة الثانية

الحد الأول = س٤ = ( س٢ )٢ مربع كامل .

الحد الثالث = ٩ = ( ٣ )٢ مربع كامل .

فيكون الحد الأوسط الذي يتكون مع الحدين الأول والثالث مقدارًا ثلاثيًا على صورة مربع كامل هو : ٢ × س٢ × ٣ = ٦ س٢ .

لذلك إذا أضفنا الحد ٦ س٢ ثم طرحناه من المقدار المفروض نحصل على :

س٤ + ٢ س٢ + ٩ = ( س٤ + ٦ س٢ + ٩ ) − ٦ س٢

= ( س٢ + ٣ )٢ − ( √٦ س )٢

= [ س٢ + ( ٣ + √٦ س ) ][ س٢ + ( ٣ − √٦ س ) ]

= ( س٢ + ٣ + √٦ س )( س٢ + ٣ − √٦ س )

٧ مثال

حلل المقدار : ٣٦ س٤ − ١٠٠ س٢ ص٢ + ٤٩ ص٤ .

الحل :

الحد الأول : ٣٦ س٤ = ( ٦ س٢ )٢ مربع كامل .

الحد الأخير : ٤٩ ص٤ = ( ٧ ص٢ )٢ مربع كامل .

الحد الأوسط الذي يتكون مع الحدين ٣٦ س٤ ، ٤٩ ص٤ مربعًا كاملاً هو : ٢ × ٦ س٢ × ٧ ص² = ٨٤ س٢ ص² .

∴ ٣٦ س٤ − ١٠٠ س٢ ص² + ٤٩ ص٤

= ٣٦ س٤ − ٨٤ س² ص² + ٤٩ ص⁴ − ١٠٠ س² ص² − ٨٤ س² ص²
