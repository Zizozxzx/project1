الحل :

تعلم أن مساحة الحديقة المربعة = ( طول ضلعها )٢.

∴ ( س٢ + ٢٠س + ١٠٠ ) = ( س + ١٠ )٢.

∴ طول ضلع الحديقة = ( س + ١٠ ) مترًا.

تمارين ومسائل

أكمل ما يأتي لتحصل على متساويات صحيحة :

[١] س٢ − ٣س + ٢ = ( س − .... ) ( .... − ٢ )
[٢] م٢ + ١٥ + ٦ = ( .... + م ) ( ٣ + .... )
[٣] س٢ + س − ٣٠ = ( .... + س ) ( .... − س )
[٤] م٢ + ٧م هـ + ١٠هـ = ( م + ٢هـ ) ( .... + .... )
[٥] ل٢ − ٩ل م − ١٠م = ( ل − ١٠م ) ( م + .... )
[٦] ٣س٢ + ٤س − ٤ = ( .... + ٢ ) ( .... − ٣س )
[٧] ٢م٢ + ١١م + ٤ = ( ٤ + .... ) ( .... + ٢م )
