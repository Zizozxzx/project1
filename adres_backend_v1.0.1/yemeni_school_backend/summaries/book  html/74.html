٥ مثال

حلل المقدار : س٢ − ٤س − ٧ .

الحل :

لاحظ أن هذا المقدار لا يمكن تحليله بالطرق السابقة ، ولذا نحلله بإكمال المربع :

معامل س = −٤

نصف معامل س = −٢

مربع نصف معامل س = ( −٢ )٢ = ٤

∴ س٢ − ٤س − ٧ = س٢ − ٤س + ٤ − ٤ − ٧

= ( س − ٢ )٢ − ١١

= [ √١١ + ( س − ٢ ) ][ √١١ − ( س − ٢ ) ]

= ( √١١ + س − ٢ )( √١١ − س + ٢ )

قد يصادفنا أحيانًا مقدار ثلاثي كل من حديه الأول والثالث مربع كامل ولا يمكننا تحليله بالطرق السابقة فنلجأ إلى تحليله بطريقة إكمال المربع ، والمثال التالي يوضح ذلك .
