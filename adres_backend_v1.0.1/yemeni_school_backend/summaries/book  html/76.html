تحليل المقادير الجبرية
الوحدة الثانية

= ٦ س٢ − ٧ ص٢

[ ٤ س ص + ( ٧ ص٢ − ٦ س٢ ) ] [ ٤ س ص − ( ٧ ص٢ − ٦ س٢ ) ]

= ( ٦ س٢ + ٤ س ص − ٧ ص٢ ) ( ٦ س٢ − ٤ س ص − ٧ ص٢ )

لاحظ: أننا اخترنا ( −٨٤ س٢ ص٢ ) حدًا أوسطًا للحدين الأول والثالث لكي يؤول المقدار الأصلي إلى فرق بين مربعين لكي نتمكن من متابعة التحليل.

٨ مثال

حلل : ٤ أ٤ + ٨١ ب٤ .

الحل :

٤ أ٤ = ( ٢ أ٢ )٢ ، ٨١ ب٤ = ( ٩ ب٢ )٢

الحد الأوسط الذي يكون مع الحدين ٤ أ٤ ، ٨١ ب٤ مربعًا كاملاً هو : ٢ × ٢ أ٢ × ٩ ب² = ٣٦ أ² ب²

∴ ٤ أ٤ + ٨١ ب٤ = ٤ أ٤ + ٣٦ أ² ب² + ٨١ ب⁴ − ٣٦ أ² ب²

= ( ٢ أ² + ٩ ب² )² − ( ٦ أ ب )²

[ ٢ أ² + ٩ ب² + ٦ أ ب ] [ ٢ أ² + ٩ ب² − ٦ أ ب ]

تمارين ومسائل

[ ١ ] أكمل كل مقدار فيما يأتي إلى مربع كامل :

( ١ ) ب٢ + ٤ ب     ( ٢ ) س٢ − ١٠ س
