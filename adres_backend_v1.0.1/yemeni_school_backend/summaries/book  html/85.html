الوحدة الثانية

حل آخر :

( ١ + س٢ ) + ( س + س٣ ) = ١ + س + س٢ + س٣

ونترك تكملة الحل كنشاط للطالب .

مثال ( ٢ )    حلل المقدار : ٨ س٢ + ٣ س ص − ٦ ص − ١٦ س


الحل :

( ٣ س ص − ٦ ص ) + ( ٨ س٢ − ١٦ س ) = ٨ س٢ + ٣ س ص − ٦ ص − ١٦ س

٣ ص ( س − ٢ ) + ٨ س ( س − ٢ )

( ٣ ص + ٨ س ) ( س − ٢ )

مثال ( ٣ )    حلل المقدار : ص٤ + ص٣ − ص٢ − ص


الحل :

( ص٤ + ص٣ ) − ( ص٢ + ص ) = ص٤ + ص٣ − ص٢ − ص

ص٣ ( ص + ١ ) − ص ( ص + ١ )

( ص٣ − ص ) ( ص + ١ )

ص ( ص٢ − ١ ) ( ص + ١ )

ص ( ص + ١ ) ( ص − ١ ) ( ص + ١ )

ص ( ص + ١ )٢ ( ص − ١ )
