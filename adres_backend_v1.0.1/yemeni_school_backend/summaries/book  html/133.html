شكل (٣-٥)

إذن حل المعادلتين هو الزوج المرتب ( ١ ، ٢ ) .
أي أن مجموعة الحل = { ( ١ ، ٢ ) } .
التحقق :
تحقق من صحة الحل بالتعويض عن الزوج المرتب ( ١ ، ٢ ) في كلا المعادلتين المعطاتين .
