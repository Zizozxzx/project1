الوحدة الأولى
الحل :

(أ) نلاحظ أن س هي مجموعة الحروف الهجائية العربية، ونكتب ذلك بالصفة المميزة رمزياً على النحو التالي:

س = { م : م أحد الحروف الهجائية العربية }

(ب) نلاحظ أن الصفة المميزة لعناصر المجموعة ص هي: أعداد صحيحة محصورة بين ٢ و٦، أو أرقام العدد ٥٣... إلخ.

نكتفي في الحل بذكر صفة واحدة فقط من الصفات المميزة، فنكتب مثلاً:

ص = { م : م ∈ ص ، ٢ < م < ٦ }

(ج) ع = { ب : ب عددًا زوجيًا طبيعيًا، ب < ٥ }

تمارين ومسائل
  1. اكتب كلاً من المجموعات الآتية بالصفة المميزة رمزياً:
س = { محرم، صفر، ربيع أول، ... ، ذي الحجة }
ص = { ١١ ، ١٣ ، ١٥ ، ١٧ ، ١٩ }
ل' = { ١٠ ، ١١ ، ١٢ ، ١٣ ، ١٤ }
م = { ل ، ع ، ب }
  1. اكتب بذكر الصفة المميزة رمزياً كلاً من المجموعات التالية:
    (أ) مجموعة عواصم العالم    (ب) مجموعة مضاعفات العدد ٩
