الحد الأول = س٢

∴ المقدار هو : س٢ + ١٠س ص + ٢٥ص٢

(د) الجذر التربيعي للحد الثالث = −١٤م ن ÷ (٢ × م)

= −٧ن

∴ الحد الثالث = (−٧ن)٢ = ٤٩ن٢

∴ المقدار هو : م٢ − ١٤م ن + ٤٩ن٢

مثال (٨)

حل ما يأتي :

(أ) هـ٢ − ١٢هـ + ٣٦ ، (ب) ٤ل٢ + ١٢ل م + ٩م٢

(ج) م٢ − ٢م √٧ + ٧ ، (د) س٢ + ٥س + ٢٥ / ٤ص٢

الحل :

(أ) هـ٢ − ١٢هـ + ٣٦ = (هـ − ٦)٢

(ب) ٤ل٢ + ١٢ل م + ٩م٢ = (٢ل + ٣م)٢

(ج) م٢ − ٢م √٧ + ٧ = (م − √٧)٢

(د) س٢ + ٥س + ٢٥ / ٤ص٢ = ( س + ٥ / ٢ص ) ٢

مثال (٩)

حديقة أطفال مربعة الشكل مساحتها ( س٢ + ٢٠س + ١٠٠ ) مترًا مربعًا ، حيث س ≥ صفر . أوجد طول ضلع هذه الحديقة بدلالة س .
