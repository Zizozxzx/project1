تحليل المقادير الجبرية
الوحدة الثانية

إذن : س٢ + ٢ب س + ب٢ = ( س + ب )٢.

أما المقدار الثاني فيُحل على النحو التالي : س٢ + ٤س = س ( س + ٤ ).

كما نستطيع تحليله بطريقة أخرى وباستخدام خواص جبرية لإكمال المربع، فمثلاً : س٢ + ٤س لا يمثل مربعًا كاملًا، ولكي يصبح مربعًا كاملًا يجب أن نضيف حدًا، وليكن ب٢، بحيث يكون الحد الأوسط ٤س = ٢ب س، ويتحقق ذلك إذا كان ب = ٢.

لاحظ أن العدد ٢ هو نصف معامل س في هذا المقدار، وبإضافة (٢)٢ إلى س٢ + ٤س يكون لدينا س٢ + ٤س + ٤ = ( س + ٢ )٢.

لإكمال المقدار س٢ + ب س إلى مربع كامل، نضيف إليه مربع نصف معامل س، أي ( ب / ٢ )٢، فنحصل على :
س٢ + ب س + ( ب / ٢ )٢ = ( س + ب / ٢ )٢ وهو مربع كامل.
١ مثال

أكمل المقدار : س٢ + ١٤س إلى مربع كامل.

الحل :

نريد أن نكمل المربع فقط، ولذا يتغير لدينا المقدار، ولإكمال المقدار س٢ + ١٤س إلى مربع كامل نضيف مربع نصف معامل س، أي ( ١٤ / ٢ )٢ = ٤٩.

∴ س٢ + ١٤س + ٤٩ = ( س + ٧ )٢.
