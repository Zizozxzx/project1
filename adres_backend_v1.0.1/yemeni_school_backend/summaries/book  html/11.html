[ ١ ] إذا كانت : أ = { ٢ ، ٣ } ، ب = { ٣ ، ٤ } ، ج = { ٤ ، ٥ } .

أوجد كلاً مما يلي ومثله بأشكال فن :

أولاً : أ / ب    ثانياً : ب / ج    ثالثاً : ج / ب .

[ ٢ ] إذا كانت : س = { ٢ ، ٣ ، ٤ ، ٥ ، ٦ ، ٧ } ، ص = { ٣ ، ٤ ، ٨ } .

أوجد ما يلي :

( أ ) س / ص    ( ب ) ص / س

( ج ) س ∪ ( ص / س )    ( د ) س ∩ ( ص / س )

( هـ ) س ∪ ( س / ص )    ( و ) س ∩ ( س / ص )

[ ٣ ] إذا كانت المجموعة الشاملة هي مجموعة الأرقام في النظام العشري ، ما متممة مجموعة أرقام العدد ٢٩٩٧٣٥ ؟

[ ٤ ] مستعيناً بالشكل ( ٩ - ١ ) :

شكل ( ٩ - ١ )

أوجد كلاً مما يلي :

( أ ) ش̅    ( ب ) ص̅    ( ج ) ( س ∩ ص )̅

( د ) ( س ∪ ص )̅    ( هـ ) ش / س

[ ٥ ] إذا كانت : ش = { م : م ∈ ط ، ٩ > م } ، س = { ب : ب عامل من عوامل العدد ٦ } ، ص = { ١ ، ٢ ، ٧ } .

اكتب ش ، س ، ص بطريقة السرد ، ثم أوجد :

( أ ) س̅̅    ( ب ) ص̅    ( ج ) س̅̅ ∪ ص̅
