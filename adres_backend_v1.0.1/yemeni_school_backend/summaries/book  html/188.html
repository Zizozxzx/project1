[ ١٩ ] إذا كان ظا هـ = ١٠٢١ ، حيث هـ زاوية حادة، أثبت أن :

١جتا² هـ = ١ + ظا² هـ .

[ ٢٠ ] إذا كان : ٢ ظا هـ = ٣ جا هـ ، حيث هـ زاوية حادة، أوجد كلاً من : ظا هـ ، جا هـ .

[ ٢١ ] أوجد قيمة كل من :

( أ ) جتا ٦٠° − جا² ٤٥° .

( ب ) ٢ جا ٣٠° + ٢ جتا² ٣٠° − ١٢ ظا² ٦٠° .

( ج ) ( ١ + ظا² ٣٠° ) ( جا² ٦٠° − ظا² ٦٠° ) .

( د ) جتا ٦٠° ١ − ٢ جتا² ٣٠° + ٢ ظا ٤٥° .

[ ٢٢ ] أثبت أن :

١ + جتا ٦٠° ١ − جتا ٦٠° = ٢ ( ١جا ٦٠° + ١ظا ٦٠° ) .

[ ٢٣ ] أثبت أن : جا² هـ + جا هـ جتا² هـ = جا هـ .
