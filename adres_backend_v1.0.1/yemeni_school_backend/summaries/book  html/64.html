نحلل العدد (−١٢) إلى عاملين مجموعهما يساوي معامل الحد الأوسط (−٤) ، العاملان هما : −٦ ، ٢ .

نكتب المقدار بحيث يظهر معامل الحد الأوسط على صورة مجموع العاملين ، وذلك على النحو التالي :

٣س٢ − ٤س − ٤ = ٣س٢ − ٦س + ٢س − ٤

( بأخذ العامل المشترك لكل حدين متتالين على حدة )

= ٣س ( س − ٢ ) + ٢ ( س − ٢ )

= ( س − ٢ ) ( ٣س + ٢ )

التحقق :

اضرب المقدارين ( س − ٢ ) ، ( ٣س + ٢ ) .

مثال (٥)

حلل المقدار : ٢س٢ + ٧س + ٦ .

الحل :

حاصل ضرب معامل س٢ في الحد المطلق = ١٢ .

نحلل العدد ١٢ إلى عاملين مجموعهما يساوي ٧ ( معامل الحد الأوسط ) فنجد أنهما : ٤ ، ٣ .

∴ ٢س٢ + ٧س + ٦ = ٢س٢ + ٤س + ٣س + ٦

= ٢س ( س + ٢ ) + ٣ ( س + ٢ )

= ( س + ٢ ) ( ٢س + ٣ )

التحقق : تحقق بنفسك من صحة التحليل .
