الحل :
(١) نكتب إحدى المعادلتين ولتكن المعادلة (١) بدلالة أحد المتغيرين.
وليكن س فتصبح المعادلة : ص = ٢س − ٥ ................. (٣)
(٢) نعوض عن المتغير ص [ المعادلة (٣) ] في المعادلة (٢)
س + ٣ (٢س − ٥) = −١
س + ٦س − ١٥ = −١
−٧س − ١٥ + ١٥ = −١ + ١٥
٧س = ١٤
٧س ٧ = ١٤ ٧
∴ س = ٢
(٣) نعوض عن قيمة المتغير س في المعادلة (٣) فنحصل على :
ص = (٢ × ٢) − ٥
ص = ٤ − ٥
ص = −١
∴ مجموعة حل المعادلتين هي : { (٢ ، −١) }
(٤) تحقق من صحة الحل بالتعويض عن (٢ ، −١) في كلا المعادلتين (١) ، (٢) كما يلي :
المعادلة (٢) :
الطرف الأيمن = س + ٣ص = ٢ + ٣ (−١) = ٢ − ٣ = −١
∴ الطرف الأيمن = الطرف الأيسر
