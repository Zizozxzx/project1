المجموعات والعلاقات
ماذا تمثل المنطقة المظللة في شكل (١-٣)؟
شكل (١-٣)
إنها تمثل مجموعة تتكون من العناصر التي تنتمي إلى س ولا تنتمي إلى ص، وتمثل هذه المجموعة المجموعة س فرق المجموعة ص.
وتسمى المجموعة س فرق المجموعة ص، ويرمز لها بالرمز ( س / ص )، حيث أن:
{ أ : أ ∈ س ، أ ∉ ص } = س / ص
أي أن: { ٦ ، ٧ ، ٨ } = س / ص
وبالمثل ماذا تمثل المنطقة المظللة في شكل (١-٤)؟
شكل (١-٤)
إنها تمثل مجموعة العناصر التي تنتمي إلى ص ولا تنتمي إلى س، فهل تمثل ص / س؟
حيث أن:
{ أ : أ ∈ ص ، أ ∉ س } = ص / س
أي أن: { ١ ، ٢ ، ٣ } = ص / س
المجموعة س فرق المجموعة ص، هي مجموعة عناصرها تنتمي إلى المجموعة س ولا تنتمي إلى المجموعة ص، ويرمز لها بالرمز: س / ص

{ أ : أ ∈ س ، أ ∉ ص } = س / ص
مثال (١)
إذا كانت س = {٢ ، ٤ ، ٦ ، ٨} ، ص = {٨ ، ١٠ ، ١٢ ، ١٤}

أوجد:
(أ) س / ص ، ومثلها بأشكال فن.
(ب) ص / س ، ومثلها بأشكال فن.
