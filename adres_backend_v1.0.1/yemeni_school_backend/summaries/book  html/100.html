مثال (٢)
احسب في أبسط صورة :

٦ / (٩ − س²) + ١ / (٣ − س) − (س + ٤) / (س + ٣)
الحل :
٦ / ((٣ + س)(٣ − س)) + ١ / (٣ − س) − (س + ٤) / (٣ + س)
م . م . أ للمقامات = (٣ − س)(٣ + س)
= ٦ / (٩ − س²) + ١ / (٣ − س) − (س + ٤) / (٣ + س)
= ( ٦ + (٣ + س) − (س + ٤)(٣ − س) ) / ( (٣ + س)(٣ − س) )
= ( س² − ٩ ) / ( (٣ + س)(٣ − س) )
= ١
تذكر عند إجراء العمليات نتبع التسلسل التالي :
أولاً : نجري العمليات التي في الأقواس .
ثانياً : نجري عمليتي الضرب أو القسمة أيهما يسبق ( أي العملية التي تأتي أولاً على اليمين ) .
ثالثاً : نجري عمليتي الجمع أو الطرح أيهما يسبق ( أي العملية التي تأتي أولاً على اليمين ) .
