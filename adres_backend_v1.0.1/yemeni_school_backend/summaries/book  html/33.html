مثال ( ٤ ) إذا كانت ك = { ١ ، ٢ ، ٣ } ، ل = { ١ ، ٣ ، ٥ ، ٧ } ،
وكان ت : ك ← ل معرفاً بالقاعدة : ت ( س ) = ٢س + ١
أ ) اكتب صورة كل عنصر ، ثم عيّن المدى .
ب ) مثّل التطبيق سهمياً وبيانياً .
الحل :
أ ) المجال = ك = { ١ ، ٢ ، ٣ } ، المجال المقابل = ل = { ١ ، ٣ ، ٥ ، ٧ } .
∵ ت ( س ) = ٢س + ١ ، منها :
ت ( ١ ) = ٢ × ١ + ١ = ٣
ت ( ٢ ) = ٢ × ٢ + ١ = ٥
ت ( ٣ ) = ٢ × ٣ + ١ = ٧
∴ مدى التطبيق = { ٣ ، ٥ ، ٧ } .
ب ) الشكل ( ١ - ٢٧ أ ) يمثل التطبيق سهمياً والشكل ( ١ - ٢٧ ب ) يمثله بيانياً .
