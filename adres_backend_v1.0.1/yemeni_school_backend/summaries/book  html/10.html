أوجد : ( أ ) س̅ ، ( ب ) س̅̅ ، ( ج ) قارن المجموعتين س ، س̅ .

ماذا تلاحظ ؟

نلاحظ أن : س̅̅ = س

أي أن : متممة المتممة للمجموعة س هي المجموعة س نفسها .

ثالثاً : قانونا دي مورجان :

نشاط

إذا كانت ش = { ١ ، ٢ ، ٣ ، ٤ ، ٥ ، ٦ } ، س = { ١ ، ٢ ، ٣ } ، ص = { ٣ ، ٤ ، ٥ } ، أوجد :

( أ ) س ∩ ص    ( ب ) س ∪ ص    ( ج ) س̅

( د ) ص̅    ( هـ ) س̅ ∩ ص̅    ( و ) س̅ ∪ ص̅

( ز ) ( س ∪ ص )̅    ( ح ) ( س ∩ ص )̅

( ط ) قارن ( هـ ) ، ( ز ) وكذلك ( و ) ، ( ح ) . ماذا تلاحظ ؟

بمقارنة إجابتي ( هـ ) ، ( ز ) نلاحظ أنهما متساويتان ، وبمقارنة إجابتي ( و ) ، ( ح ) نلاحظ أنهما متساويتان أيضاً .

أي أن :

( ١ ) ( س ∪ ص )̅ = س̅ ∩ ص̅
( ٢ ) ( س ∩ ص )̅ = س̅ ∪ ص̅

ويُسمى هذان القانونان بقانوني دي مورجان .
