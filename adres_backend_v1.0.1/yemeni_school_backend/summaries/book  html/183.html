حساب المثلثات الوحدة الرابعة

جا ٤٥° = ١√٢

جتا ٤٥° = ١√٢

ظا ٤٥° = ١

مثال ( ١ ) أوجد قيمة كل مما يلي :

( أ )   جا ٦٠° + ٥ جتا ٣٠°

( ب )   ٤ جا ٣٠° − ٣ ظا ٤٥°

الحل :

( أ )   جا ٦٠° + ٥ جتا ٣٠° = √٣٢ + ( √٣٢ × ٥ ) = ٣√٣٢

( ب )   ٤ جا ٣٠° − ٣ ظا ٤٥° = ( ٤ × ١٢ ) − ( ٣ × ١ ) = ٢ − ٣ = −١

مثال ( ٢ ) أثبت أن :

١ جتا² ٦٠° = ١ + ظا² ٦٠°

البرهان :

الطرف الأيمن : ١ + ظا² ٦٠° = ١ + ( √٣ )² = ٤   ← ( ١ )

الطرف الأيسر : ١ جتا² ٦٠° = ١ ( ١ / ٢ )² = ٤   ← ( ٢ )

بمقارنة المعادلتين ( ١ ) ، ( ٢ ) نحصل على :

١ جتا² ٦٠° = ١ + ظا² ٦٠° وهو المطلوب .
