المجموعات والعلاقات
٣ . العلاقة المتعدية

تذكّر :

العلاقة ع على المجموعة س هي مجموعة جزئية من حاصل ضرب المجموعتين س × س .

أي أن : ع ⊆ س × س .

العلاقة الانعكاسية :

تكون العلاقة ع انعكاسية على المجموعة س ، إذا كان لكل أ ∈ س فإن ( أ ، أ ) ∈ ع .

العلاقة المتناظرة :

تكون العلاقة ع متناظرة على المجموعة س ، إذا كان لكل ( أ ، ب ) ∈ ع ، فإن ( ب ، أ ) ∈ ع ، حيث أ ، ب ∈ س .

مثال ( ١ )

إذا كانت : س = { ٤ ، ٥ ، ٦ } ، بيّن نوع العلاقات التالية ( انعكاسية ، متناظرة ) مع ذكر السبب :

( أ ) ع₁ = { ( ٦ ، ٦ ) ، ( ٥ ، ٥ ) ، ( ٤ ، ٤ ) ، ( ٥ ، ٤ ) } .

( ب ) ع₂ = { ( ٥ ، ٦ ) ، ( ٤ ، ٤ ) ، ( ٦ ، ٥ ) ، ( ٥ ، ٥ ) } .

( ج ) ع₃ = { ( ٥ ، ٥ ) ، ( ٤ ، ٤ ) ، ( ٤ ، ٥ ) ، ( ٦ ، ٦ ) ، ( ٥ ، ٤ ) } .

( د ) ع₄ = { ( ٥ ، ٥ ) ، ( ٤ ، ٤ ) ، ( ٤ ، ٥ ) } .

الحل :

( أ ) ع₁ انعكاسية لأن كل عنصر في س ارتبط بنفسه .

ع₁ ليست متناظرة ، لأن ( ٥ ، ٤ ) ∈ ع₁ ، ولكن ( ٤ ، ٥ ) ∉ ع₁ .
