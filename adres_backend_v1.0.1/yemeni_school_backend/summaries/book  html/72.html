الوحدة الثانية
٢ مثال

أكمل المقدار : س٢ − ٩س إلى مربع كامل.

الحل :

نضيف مربع نصف معامل س لكي يتحول المقدار إلى مربع كامل.

∴ س٢ − ٩س + ( ٩ / ٢ )٢ = ( س − ٩ / ٢ )٢.

ولتحليل المقدار س٢ + ٤س باستخدام طريقة إكمال المربع نضيف مربع نصف معامل س ثم نطرحه للمحافظة على قيمة المقدار كما يلي :

س٢ + ٤س + ٢٢ − ٤ = ( س + ٢ )٢ − ٤

= [ ٢ + ( س + ٢ ) ][ ٢ − ( س + ٢ ) ]

= س ( س + ٤ ).

في الأمثلة الآتية سوف نوضح طريقة تحليل المقادير الثلاثية بإكمال المربع.
