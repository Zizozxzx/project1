المجموعات والعلاقات

وتمثيلها بالمخطط السهمي في الشكل ( ١٧ - ١ ) .

شكل ( ١٧ - ١ )

مثال ( ٢ )

لتكن م = { ٣ ، ٤ ، ٥} ، ع علاقة على م حيث

ع = { ( ٥ ، ٥ ) ، ( ٤ ، ٤ ) ، ( ٣ ، ٤ ) ، ( ٤ ، ٣ ) ، ( ٣ ، ٣ ) } .

هل ع علاقة تكافؤ ؟ ولماذا ؟

الحل :

ع انعكاسية لأن لكل أ ∈ م فإن ( أ ، أ ) ∈ ع .

ع متناظرة لأن ( ٣ ، ٤ ) ∈ ع وأيضاً ( ٤ ، ٣ ) ∈ ع .

∵ ( ٣ ، ٤ ) ، ( ٤ ، ٣ ) ∈ ع نجد أن ( ٣ ، ٣ ) ∈ ع .

وكذلك ( ٤ ، ٣ ) ، ( ٣ ، ٤ ) ∈ ع نجد أن ( ٤ ، ٤ ) ∈ ع .

∴ ع علاقة متعدية .

∴ ع علاقة انعكاسية ومتناظرة ومتعدية .

∴ ع علاقة تكافؤ .

مثال ( ٣ )

إذا كانت ك = { ٣ ، ٥ ، ٧ } ، وكانت ع علاقة على ك حيث

ع = { ( ٣ ، ٣ ) ،( ٣ ، ٥ ) ،( ٥ ، ٥ ) ، ( ٧ ، ٧ ) } .
