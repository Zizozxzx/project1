تحليل المقادير الجبرية
مثال ( ١ )
اكتب كلا من الكسور الآتية في أبسط صورة :

( أ ) ٣ س٢ ص ٦ س ص٣

( ب ) س٢ − ٤ س٢ + ٣ س − ١٠

( ج ) ٢ س٣ − ٦ س٢ + ٤ س ٦ س٣ + ٦ س٢ − ٣٦ س

الحل :

( أ ) ٣ س٢ ص ٦ س ص٣ = س ٢ ص٢

( ب ) س٢ − ٤ س٢ + ٣ س − ١٠ = ( س + ٢ )( س − ٢ ) ( س + ٥ )( س − ٢ )

= س + ٢ س + ٥

( ج ) ٢ س ( س٢ − ٣ س + ٢ ) ٦ س ( س٢ + س − ٦ )

= ٢ س ( س − ١ )( س − ٢ ) ٦ س ( س + ٣ )( س − ٢ )

= س − ١ ٣ ( س + ٣ )
