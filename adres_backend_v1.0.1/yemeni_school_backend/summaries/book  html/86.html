تحليل المقادير الجبرية
الحل :

نلاحظ أن المقدار مكوّن من خمسة حدود كما تلاحظ أن :

س٢ − ٤ س + ٤ تمثل مربعًا كاملًا ولذلك يتم التجميع للحدود على النحو التالي :

( س٢ − ٤ س + ٤ ) + ( ٢ ل س − ٤ ل ) = س٢ − ٤ س + ٢ ل س − ٤ ل + ٤

( س − ٢ )٢ + ٢ ل ( س − ٢ )

[ ٢ ل + ( س − ٢ ) ] ( س − ٢ )

( س − ٢ + ٢ ل ) ( س − ٢ )

تمارين ومسائل

حل كلًا مما يلي :

[ ١ ]   م س + ب س + م ص + ب ص .

[ ٢ ]   أ ب − ب س + هـ − أ هـ .

[ ٣ ]   س ص + ٥ ص + ٧ س + ٣٥ .

[ ٤ ]   أ٢ ب − ١ − أ٢ ب .

[ ٥ ]   ٢ م٢ ص + ٤ ص − ص − ٢ م .

[ ٦ ]   ٣ س٢ + ٢ س + ٨ + ١٢ س .

[ ٧ ]   ٣٤ م٢ + ١٢ م − ٩ − م٢ .

[ ٨ ]   ٢٥ س٢ + ٤٠ س ص + ١٦ ص٢ + ٥ س + ٤ ص .

[ ٩ ]   ٢ م − ٣ ب + ٤ ب٢ − ١٢ ب + ٩ ب٢ .

[ ١٠ ]   ٦ س − ١٠ س ص + ١٢ ص − ٢٠ ص٢ .

[ ١١ ]   س٢ + ٧ س − ٣ س٢ − ٢١ .

[ ١٢ ]   ١٢ ب٢ − ٧٢ ب + ٤٨ + ب٢ .
