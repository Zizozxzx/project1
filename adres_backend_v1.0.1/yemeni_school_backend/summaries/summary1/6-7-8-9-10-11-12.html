الفرق بين مجموعتين ومتممة ( مكملة ) مجموعة
( ١ ) الفرق بين مجموعتين : الفرق بين مجموعتين س ، ص هو مجموعة مكونة من مجموعة العناصر التي تنتمي إلى س ولا تنتمي إلى ص ونرمز لها بالرمز س / ص ونكتب س / ص = { أ : أ ∈ س ، أ ∉ ص } .
( ٢ ) متممة ( مكملة ) مجموعة : إذا كانت ش هي المجموعة الشاملة فإن متممة المجموعة س هي مجموعة العناصر التي تنتمي إلى المجموعة الشاملة ش ولا تنتمي إلى س ويرمز لها بالرمز س' ونكتب ذلك : س' = ش / س أي أن س' = { س : س ∈ ش ، س ∉ س } .
( ٣ ) متممة متممة المجموعة : هي المجموعة س نفسها أي ( س' )' = س .
( ٤ ) قانوني دي مورجان هما : ١) ( س ∪ ص )' = س' ∩ ص'    ٢) ( س ∩ ص )' = س' ∪ ص' .
