أولاً : الفرق بين مكعبين :
الفرق بين مكعبي حدين =
(الجذر التكعيبي للأول - الجذر التكعيبي للثاني)
× ( مربع الأول + الأول × الثاني + مربع الثاني )

أي أن : أ3 - ب3 = ( أ - ب ) ( أ2 + أ ب + ب2 ) " قانون "

ثانياً : مجموع مكعبين :
مجموع مكعبي حدين =
( الحد الأول + الحد الثاني ) ( مربع الأول
- ضرب الحدين + مربع الحد الثاني )

و) 27 ع3 + 25 لا يمثل مجموع مكعبين لأن 25 ليس مكعب كامل أي لا يمكن تحليله إلى ثلاثة عوامل متساوية .

ملاحظة :
حاصل ضرب قوتين متفقتين بالأساس : أم × أن = أم+ن فمثلاً :
