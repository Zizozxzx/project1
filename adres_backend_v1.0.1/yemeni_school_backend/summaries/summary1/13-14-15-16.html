* حل رقم ٥ : *

ش = { ٠ ، ١ ، ٢ ، ٣ ، ٤ ، ٥ ، ٦ ، ٧ ، ٨ } ، س = { ١ ، ٢ ، ٣ ، ٦ } ، ص = { ٢ ، ٧ }

(أ) سَ = { ٠ ، ٤ ، ٥ ، ٧ ، ٨ }

(ب) صَ = { ٤ ، ٣ ، ٠ ، ١ ، ٥ ، ٦ ، ٨ }

(ج) سَ U صَ = { ٠ ، ١ ، ٣ ، ٤ ، ٥ ، ٦ ، ٧ ، ٨ }

(د) ∵ (س ∩ ص) = { ٢ ، ٧ }
∴ (س ∩ ص)َ = { ٠ ، ١ ، ٣ ، ٤ ، ٥ ، ٦ ، ٨ } = سَ U صَ

(هـ) سَ / صَ = { ٧ } = { ٠ ، ٤ ، ٣ ، ٤ } / { ٨ ، ٧ ، ٦ ، ٥ ، ٤ ، ٣ }

[٦] إذا كانت ش = { ١ ، ٢ ، ٣ ، ٤ ، ٥ } ، س = { ١ ، ٢ } ، ص = { ٤ ، ١ ، ٥ } ، ع = { ٢ ، ٣ } فأوجد :-

أ) سَ / ص
∵ سَ = { ٣ ، ٤ ، ٥ } ∴ سَ / ص = { ٣ ، ٤ ، ٥ } / { ١ ، ٤ ، ٥ } = { ٣ }

ب) ع / س = { ٢ ، ٣ } / { ١ ، ٢ } = { ٣ }

ج) سَ / صَ = { ٣ ، ٤ ، ٥ } / { ٢ ، ٣ } = { ٤ ، ٥ }

د) (س / ص) ∩ (ع / صَ)
صَ = { ٢ ، ٣ ، ٥ }
نجد أولاً: س / ص = { ٢ } ، ع / صَ = { ∅ }
∴ الناتج = { ٢ } ∩ { ∅ } = ∅

العلاقــــــات
تذكير :

[١] الزوج المرتب : يسمى العنصر (أ ، ب) زوج مرتب ويسمى العنصر الأول أ المسقط الأول والعنصر الثاني ب يسمى المسقط الثاني.

[٢] تساوي زوجان مرتبان : يقال أن (أ ، ب) = (جـ ، د) إذا كان أ = جـ ، ب = د .

[٣] حاصل ضرب مجموعتين س ، ص : هو مجموعة كل الأزواج المرتبة التي مسقطها الأول ينتمي إلى س ومسقطها الثاني ينتمي إلى ص ونكتب س × ص = { (أ ، ب) : أ ∈ س ، ب ∈ ص } وعـدد هذه الأزواج = حاصل ضرب عناصر س في عناصر ص.

[٤] العلاقة : العلاقة من مجموعة غير خالية س إلى مجموعة غير خالية ص : هي مجموعة جزئية من حاصل ضرب المجموعتين س × ص ونرمز لذلك ع ⊆ س × ص حيث نكتب العلاقة ع بالصورة التالية : ع = { (أ ، ب) : أ ∈ س ، ب ∈ ص }.

[٥] العلاقة من مجموعة س إلى نفسها : هي مجموعة جزئية من س × س أي ع ⊆ س × س وتكتب ع = { (أ ، ب) : أ ، ب ∈ س }.

ملاحظة : الزوج المرتب (أ ، ب) ∈ ع يعني به أن العنصر أ من س يرتبط بالعنصر ب من ص حيث كل عنصر (أ ، ب) ∈ ع ينتمي إلى س × ص .

مثال: إذا كانت س = { ١ ، ٢ ، ٣ } ، ص = { -١ ، ٢ } فأوجد س × ص ثم اكتب العلاقات التالية من س إلى ص :-

أ) ع١ = { (س ، ص) : س < ص ، س ∈ س ، ص ∈ ص } (علاقة أصغر من) .

ب) ع٢ = { (س ، ص) : س > ص ، س ∈ س ، ص ∈ ص } (علاقة أكبر من) .

ثم ارسم المخطط السهمي والبياني لكل علاقة.
