المقدار الثلاثي
الصورة العامة للمقدار الثلاثي هي :
أ س٢ + ب س + جـ    ، أ ≠ ٠ ، أ ، ب ، جـ ∋ ح
وهما نوعان :-
١ | مقدار ثلاثي بسيط صورته العامة هي : س٢ + ب س + جـ
حيث معامل س٢ هو ١ = أ ، ب ، جـ ∋ ح
ب | مقدار ثلاثي غير بسيط صورته العامة هي : أ س٢ + ب س + جـ
حيث أ ≠ ٠ ، أ ≠ ١ ، أ ، ب ، جـ ∋ ح
* أولاً : تحليل المقدار الثلاثي البسيط
الذي صورته : س٢ + ب س + جـ
لتحليل المقدار الثلاثي البسيط نتبع ما يلي :-
نبحث عن عددين نفرض أنهما م ، ن بحيث يكون ( أي نحلل الحد المطلق جـ إلى عاملين م ، ن ) بحيث يكون :
مجموعهما م + ن = ب ( معامل س للحد الأوسط )
وحاصل ضربهما م × ن = جـ ( الحد الثالث المطلق ) فيكون :
س٢ + ب س + جـ = س٢ + ( م + ن ) س + م ن
               = س٢ + م س + ن س + م ن
               = س ( س + م ) + ن ( س + م )
               = ( س + م ) ( س + ن ) # التحليل المطلوب
مثال : حلل المقدار س٢ + ٣ س + ٢
( من خلال هذا المثال سنوضح بعض الطرق في الحل )
* الحل *
|١| الطريقة الأولى: تلاحظ أن : الحد الأول س٢ = س × س
الحد الأوسط = ٣ س والحد المطلق (الثالث) = ٢ (موجب)
لذلك عاملاه كلاهما موجبان أو سالبان معاً نكتب عوامله في الجدول المجاور ثم نبحث عن عاملين للعدد (٢) بحيث يكون :
حاصل ضربهما = ٢
ومجموعهما = ٣ معامل الحد الأوسط س فنجد أنهما ٢ ، ١ فيكون التحليل
المطلوب هو : س٢ + ٣ س + ٢ = ( س + ١ ) ( س + ٢ )
عوامل العدد ٢مجموع العاملين
٢ × ١٢ + ١ = ٣
-٢ × -١-٢ - ١ = -٣
|٢| الطريقة الثانية : طريقة المقص : تتم كالتالي :
أ) نرسم خطين متقاطعين كما هو موضح في الشكل المقابل.
ب) نحلل الحد الأول على يمين الخطين ونحلل الحد المطلق على يسارهما. وتوضع الإشارة في وسط الحدين من الوسطين أعلى ومن أسفل .
جـ) نحصل على الحد الأوسط من المجموع الجبري لعاملي حاصل ضرب الطرفين والوسطين
أي الحد الأوسط :
س × ٢ + س × ١ = ٢ س + س = ٣ س
فيكون لتحليل المطلوب : س٢ + ٣ س + ٢ = ( س + ١ ) ( س + ٢ )
|٣| الطريقة الثالثة : لتحليل المقدار : س٢ + ٣ س + ٢ نتبع ما يلي :
أ) نجد حاصل ضرب الحدين الأول والأخير = س٢ × ٢ = ٢ س٢
ب) نحلل ناتج الضرب في (أ) إلى عاملين مجموعهما = الحد الأوسط = ٣ س فنجد أنهما ٢ س ، س أي ٢ س + س = ٣ س
جـ) نكتب المقدار الأصلي واضعين العاملين بدلاً عن الحد الأوسط
أي : س٢ + ٣ س + ٢ = س٢ + (٢ س + س) + ٢ نأخذ العامل المشترك الأكبر لكل حدين متتاليين على حده
= س(س + ٢) + ١(س + ٢)
نأخذ المقدار المشترك ( س + ٢ )
فنحصل على التحليل المطلوب = ( س + ١ ) ( س + ٢ )
ملاحظة : للتحقق من صحة الحل اضرب المقدارين ( العاملين ) تجد المقدار الأصلي أي أن :
( س + ١ ) ( س + ٢ ) = س ( س + ٢ ) + ١ ( س + ٢ )
( نضرب كل حد من العامل الأول في المقدار ثم نجمع الحدود المتشابهة ) = س٢ + ٢ س + س + ٢ = س٢ + ٣ س + ٢
* ثانياً : تحليل المقدار الثلاثي غير البسيط
الذي صورته هي : أ س٢ + ب س + جـ  ، أ ≠ ١ ، أ ≠ ٠
أ ، ب ، جـ ∋ ح حيث أ معامل س٢
، ب معامل الحد الأوسط ، جـ حد مطلق
لتحليل المقدار الثلاثي البسيط نستخدم إحدى الطريقتين الثانية ( المقص ) أو الثالثة التي أوضحناهما سابقاً في تحليل المقدار الثلاثي البسيط كما سنريهما على المثال التالي :-
مثال : حلل المقدار ٣ س٢ + ٤ س - ٤
* الحل * الطريقة الأولى :
أ) نجد حاصل ضرب الحدين الأول والأخير أي :
٣ س٢ × ( -٤ ) = -١٢ س٢ ( ممكن نضرب معامل س٢ في الحد المطلق ثم نكتب معامل الأوسط = مجموع العاملين )
ب) نحلل ناتج الضرب في ( أ ) إلى عاملين مجموعهما = الحد الأوسط = ٤ س ( نلاحظ أن -١٢ س٢ يحلل إلى عاملين مختلفين في الإشارة ) فنجد أن العاملين هما ٦ س ، -٢ س حيث يكون ٦ س - ٢ س = ٤ س ( الحد الأوسط ) .
جـ) نكتب المقدار الأصلي واضعين مجموع العاملين بدلاً عن الحد الأوسط ثم نأخذ العامل المشترك بين كل حدين متتاليين ثم العامل المشترك بين المقادير أي :
٣ س٢ + ٤ س - ٤ = ٣ س٢ + ٦ س - ٢ س - ٤
                   = ٣ س ( س + ٢ ) - ٢ ( س + ٢ )
                   = ( ٣ س - ٢ ) ( س + ٢ )
وهو التحليل المطلوب للمقدار .
الطريقة الثانية : طريقة المقص : راجع الخطوات ( أ ) ، ( ب ) ، ( جـ ) الموضحة في المقدار البسيط
حيث نحلل الخطين نحلل الحد الأول ٣ س٢ إلى عاملين هما ٣ س ، س وكذلك نحلل الحد الثالث (-٤) إلى عاملين مختلفي الإشارة وهما ٢ ، -٢
أو ١ ، -٤ أو -١ ، ٤ ثم نتحقق من أن المجموع الجبري لحاصل ضرب الوسطين وحاصل ضرب الطرفين = الحد الأوسط للمقدار
٣ س × ٢ + س × -٢ = ٦ س - ٢ س = ٤ س = الحد الأوسط
فتجد : ٣ س٢ + ٤ س - ٤ = ( ٣ س - ٢ ) ( س + ٢ )
ملاحظة : الإشارة العليا تنتج من ضرب إشارة الحد الأول ، والإشارة السفلى تنتج من ضرب إشارة الحد الأوسط × إشارة الحد الأخير
