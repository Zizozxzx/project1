أسئلة وزارية

ظلل الإجابة الصحيحة فيما يلي:
أ. العلاقة التي تربط كل عنصر من عناصر المجال بصورته في المجال المقابل تسمى .... :
1. منطلق العلاقة.
2. مدى التطبيق.
3. قاعدة التطبيق.
4. مستقر التطبيق.
ب. لتكن س = { -1 ، 1 }، وعرف التطبيق ت : س ← ح بالعلاقة ت ( س ) = س² - 1 ، فإن المدى = .......... :
1. { -1 }.
2. { 0 ، 1 }.
3. { 0 }.
4. { -1 ، 0 }.
ج. قاعدة التطبيق التي تربط كل عنصر من مجموعة بمربع ضعفه هي ت ( س ) = ............... :
1. 2 س².
2. ( س + 2 )².
3. 4 س.
4. 4 س².
د. العلاقة من س إلى ص والتي تربط كل عنصر من س بعنصر واحد فقط من ص تسمى ............. :
1. مجموعة.
2. مخطط.
3. تطبيق.
4. شكل فن.
و. إذا كان ت: س ← ص ، معرفاً بالقاعدة ت(س) = ½ س + 1 ، حيث س= { -2 ، 2 }، فإن المدى = ........ :
1. { -1 ، 2 }.
2. { 1 ، 2 }.
3. { 0 ، 2 ، 3 }.
4. { 0 ، 2 }.
هـ. إذا كان ت: ح ← ح تطبيق معرف بالقاعدة ت ( أ ) = أ² ، فإن ت ( -1 ) = .....
1. -1.
2. 1.
3. 2.
4. -2.
لكي تصل إلى ما تريده أشعل العزيمة بداخلك
