تمارين ومسائل (صفحة ١٢٤)
[ ١ ] عين فيما يلي معادلات الدرجة الأولى في متغيرين :
الحل: (أ) ٢ س − ص = ٤ -> معادلة درجة أولى
(ب) س٢ − ٣ ص = ٧ -> ليست درجة أولى (بسبب س٢)
(ج) س ( ١ + س ص ) = ٠ -> ليست درجة أولى
(د) ٠ = ٩ + م ص + ل -> معادلة درجة أولى (في متغيرين م ص)
(هـ) ١/٤ س + ١/٣ ص = ٣٠ -> معادلة درجة أولى
(و) س٢ − ص٢ = ٤ -> ليست درجة أولى
[ ٢ ] اكتب المعادلات التالية بدلالة ص :
الحل: (أ) س + ص = ٥ => ص = ٥ − س
(ب) ص − ٤ س = ١٥ => ص = ٤ س + ١٥
(د) ٢ ص + ٣ س = −٦ => ٢ ص = − ٣ س − ٦ => ص = − ١.٥ س − ٣
