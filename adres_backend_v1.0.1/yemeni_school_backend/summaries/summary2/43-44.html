الدرس الأول: كتابة المجموعة بالصفة المميزة

يتم كتابة المجموعة بطرق مختلفة والتي تتمثل في التالي:

طريقة السرد

كتابة المجموعة بطريقة السرد تتمثل بعدة شروط هي:

  1. أ. كتابة اسم المجموعة بالرموز الكبيرة مثل: س، ص، ع، ش، هـ، ح، ع.
  2. ب. كتابة ( = ) ثم حاصرتين ( { } ).
  3. ج. كتابة العناصر بين الحاصرتين ووضع فاصلة بين كل عنصر ( ، ).
  4. د. لا يشترط ترتيب العناصر، والتكرار لها ممنوع.

مثال على ذلك : أ = { ي ، م ، ن }.
طريقة الصفة المميزة لفظياً

كتابة المجموعة بطريقة الصفة المميزة لفظياً تتمثل في كتابة الصفة المشتركة لجميع عناصر المجموعة مثل:

  • ٢، ٣، ٥، ٧،... ← الصفة المشتركة بينها هي أنها أعداد طبيعية أولية.
  • شمال، شرق، غرب، جنوب ← الصفة المشتركة بينها هي أنها تمثل الجهات الأربع الأصلية.

مثال على ذلك : ل هي مجموعة الأعداد الصحيحة المحصورة بين ٢ و ١٢.
طريقة الصفة المميزة رمزياً

كتابة المجموعة بطريقة السرد تتمثل في الصيغة التالية:

أ = { أ : أ عدد زوجي طبيعي ، أ < ٨ }

تدريب (١)
  • • اكتب ما يلي بطريقة السرد:
  • ١. س هي مجموعة الأعداد الطبيعية المحصورة بين ٣ و ١٢.
  • ٢. ص هي مجموعة الحواس الخمس.
  • ٣. ح هي مجموعة الأعداد الصحيحة الأولية الأكبر من ٢.
  • ٤. ل هي مجموعة عوامل العدد ١٦.
  • ٥. ع هي مجموعة أحرف كلمة هلال.
تدريب (٢)
  • • اكتب ما يلي بطريقة الصفة المميزة رمزياً:
  • ١. س = { ٩ ، ١٨ ، ٢٧ ، ... }
  • ٢. ص = { أ ، ح ، م ، د }
  • ٣. ح هي مجموعة أرقام العدد ١٢٣٢.
  • ٤. ل = { (٠ ، ٠) ، (١ ، ١) ، (٢ ، ٢) ، ... }
